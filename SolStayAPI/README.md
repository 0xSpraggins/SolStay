# SolStayAPI

The backend of the SolStay Application built using ExpressJS and MySql.

## Available API Calls (Currently to localhost)

#### Post

- '/user'
- '/newProperty'
- '/saveReservation'

#### Get

- '/getUser'
- '/getUserProperties'
- '/getAllProperties'
- '/getUsersReservations'
- '/getPropertyDetails'

#### Put

- '/updateUser'

#### Delete

- N/A
